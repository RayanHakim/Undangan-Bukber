# 🌙 Undangan Digital Buka Bersama MIPA 4

Repositori ini berisi kode sumber untuk microsite undangan digital acara **Silaturahmi & Buka Bersama Keluarga Besar MIPA 4**. Proyek ini didesain dengan estetika modern-islami yang bersih, menggunakan aksen warna emas (*gold*) dan fitur interaktif untuk meningkatkan antusiasme peserta.

---

## 🔗 Live Demo
Lihat undangan secara langsung di sini: [rayanhakim.github.io/Undangan-Bukber/](https://rayanhakim.github.io/Undangan-Bukber/)

---

## ✨ Fitur Unggulan
* **Modern Islamic Design:** Menggunakan pola islami halus sebagai latar belakang dan gradasi emas pada tipografi.
* **Real-time Iftar Countdown:** Fitur penghitung mundur otomatis menuju waktu berbuka puasa (Maghrib) pada hari H.
* **Integrated Background Music:** Fitur audio yang otomatis aktif ketika undangan dibuka (dengan tombol kontrol Play/Pause).
* **Interactive Map Link:** Integrasi langsung ke Google Maps untuk lokasi **Kedai Satu Satu**.
* **Smooth Animations:** Animasi elemen yang muncul secara elegan saat di-scroll menggunakan **AOS.js**.
* **Glassmorphism UI:** Desain kartu informasi menggunakan efek kaca transparan (*glass-card*) yang modern.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5 & Tailwind CSS
* **Animations:** AOS.js (Animate On Scroll)
* **Icons:** Font Awesome 6.4.0
* **Typography:** Playfair Display (Serif) & Poppins (Sans-serif) via Google Fonts
* **Scripting:** JavaScript Vanilla (untuk logika Countdown dan Audio Control)
